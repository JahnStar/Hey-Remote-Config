## Install

### via Git URL

Open `Packages/manifest.json` with your favorite text editor. Add following line to the dependencies block:
```json
{
  "dependencies": {
    "com.jahnstar.hey-remote-config": "https://github.com/jahnstar/hey-remote-config.git"
  }
}
```
